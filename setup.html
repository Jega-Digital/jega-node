How to upgrade your bot (Node.js)

To access our endpoints, wrap your standard fetch requests with the x402 SDK. This will stream micro-cents (USDC) from your wallet only when you make a successful request.

npm install @x402/sdk

Then, update your script with this 5-line wrapper:

import { x402Client, wrapFetchWithPayment, registerExactEvmScheme } from "@x402/sdk";
import { privateKeyToAccount } from "viem/accounts";
import { base } from "viem/chains";

// 1. Load your funded Base wallet
const account = privateKeyToAccount("0xYOUR_PRIVATE_KEY");
const client = new x402Client();

registerExactEvmScheme(client, { signer: account, chainId: base.id });
const fetchWithPayment = wrapFetchWithPayment(fetch, client);

// 2. Fetch the Premium Data!
const response = await fetchWithPayment("https://api.jega.digital/api/news/trend", {
    method: "GET",
});

const data = await response.json();
console.log(data);
